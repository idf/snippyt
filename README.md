# snippyt {:ledger:}
A command line snippet management for modern developers. 

# Usage
```
snip <snippet_name> --name=value
```

# Features 
* basic raw snippet 
* kwargs in cli 
* Prompt to ask you for missing entries 

# Development
* Python 3.6 (I enjoy Python 2 but I am not living in the past)
* Jinjia 2
* docopt
