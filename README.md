# snippyt {:ledger:}
A command line snippet management for modern developers. 

# Usage
```
Usage:
  snip <snippet>
  snip <snippet> -c <context>

Examples:
  snip helloworld -c "name=World;"
```

# Features 
* Basic plain text snippet 
* Kwargs in CLI 
* Show for missing entries
